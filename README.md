# SEOOptimizer

A **Streamlit**-based web application that automates SEO content creation. Given a user-provided topic, it fetches related keywords via the DataForSEO API, scrapes Google News for relevant articles, and uses Google Gemini to generate SEO-optimised webpage structures.

## Architecture Overview

```
User Topic
    → DataForSEO Labs API → keyword list
    → Jinja2 URL templates → Google News search URLs
    → Multi-threaded HTTP scraping → article URL list
    → Multi-threaded newspaper4k parsing → article texts
    → Google Gemini → 3 SEO-optimised JSON structures
    → Streamlit UI / TinyDB history
```

## Setup

### Prerequisites

- Python 3.13+
- A [DataForSEO](https://dataforseo.com/) account (Labs API access)
- A [Google Gemini API key](https://ai.google.dev/)

### Installation

1. **Clone the repository**

   ```bash
   git clone <repo-url>
   cd SEOOptimizer
   ```

2. **Create and activate a virtual environment**

   ```bash
   python -m venv .venv
   source .venv/bin/activate      # Linux / macOS
   # or: .venv\Scripts\activate    # Windows
   ```

3. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

4. **Configure environment variables**

   Copy the template and fill in your real credentials:

   ```bash
   cp .env .env.local   # or edit .env directly
   ```

   Edit `.env` with your keys:

   ```env
   GEM_API="your_gemini_api_key"
   SEO_USERNAME="your_dataforseo_email"
   SEO_PASSWORD="your_dataforseo_password"
   ```

   > **Security note:** The `.env` file is listed in `.gitignore` and will never be committed. The application falls back to built-in placeholder values if `.env` is missing, but these will not work against real API services.

## Usage

Run the Streamlit app:

```bash
streamlit run main.py
```

1. Enter a search topic in the text input.
2. Select keywords from the suggested list.
3. Click **"Confirm selection and start searching"**.
4. Wait for the pipeline to complete — the app will show progress bars and status messages.
5. View the three SEO-optimised website structures generated by Gemini.
6. Past searches are accessible in the sidebar ("Search History").
